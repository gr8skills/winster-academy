初中

充满活力，富有同情心和互联的学习者

 

row-1

AISG的初中部（6-8年级）是一个帮助学生探索和成长的地方。我们鼓励学生勇于冒险、尝试新鲜事物，发现自己生活中的热情所在。

我们注重学生个人的选择和挑战，为你的孩子提供真实的国际化学习经验，同时培养孩子的全球化意识和素养。

针对6、7、8年级学生的特点，我们初中部为学生在学术、艺术、体育、创造性和心智发展方面的需要量身订地设计相关的课程体系。通过对各年级课程结构的精心规划，在家长的期望下我们将帮助你的孩子适应年级过渡，并提供启发探索的课程。.

 

row 2

我们的课程

我们6-8年级的课程为孩子们提供富有挑战性的学习内容，培养他们的探索性、批判性和创新性学习思维，整个课程框架以培养出有活力、有同情心和互联学习的学生为大方向。

我们的课程设计是基于 AERO 和 Common Core 两大标准体系上就实际环境为学生量身设计的。Common Core and AERO 是许多国际学校普遍认可的两大标准体系。对于需要转换学校的家庭来说，这样的设计给孩子提供了熟悉和前后一致的课程教育。

我们相信学生应成为积极主动的学习者，常常独立思考周围的一切。作为框架的补充和课程内容的丰富，初中部还为学生提供一系列多姿多彩的课外活动。

初中学习科目指南

 

TARGET 与 Discovery 课程

TARGET和Discovery 是我们核心课程的两大重要而不可或缺的补充。

 

TARGET（6个英文单词的缩写，分别是Teen青少年、Action行动、Reflection反思、Growth成长、Exploration探索、Teamwork团队协作。TARGET是我们初中部各班主任带领的课程，旨在通过总体的指导和支持帮助青少年。

TARGET 课程寻求建立集体认同感，促使学生建立积极的自我形象，并确保学生间的互动联结和鼓励倡导。

Discovery课程
Discovery是一个集中的选修课程。学生可在低风险、低学习压力的环境下选择自己新领域里的兴趣课程。

Discovory课程设计精妙的设置让初中生在常规课程设置外尝试新领域知识。

Row 3



 
 

体验式教学

我们通过各式的初中课程为学生提供创造性、差异性的学习体检，确保你的孩子能获取一系列精心设计的体验式机会促进个人的优异成长。

通过和广州本地，全国性和国际性的组织机构合作，在旅程中让初中生感受所到国家的丰富文化底蕴并让学习过程更有意义。

 

 

科学城图书馆AISG科学城图书馆集合了广州最好的英语书籍资源，藏书量和流通量多达17，000本，为许多研究提供丰富的资料参考。另外，图书馆设有“创造空间”，它的设立鼓励着学生们发挥个人的创造、创新能力和学习热情。学生自主设计、以大量学生艺术品作为装饰点缀的科学城图书馆也成为了学生暂离繁杂的港湾。

我们图书馆的员工总是孜孜不倦的倡导阅读。这里有午餐时间举行的学生书友会，也赞助不同年度“读书战”竞赛的区域赛队伍在这里进行竞技。通过和老师们的紧密合作，我们的图书馆馆长将各样文学写作技能综合融入我们的学生课程当中，让所有学生的学习更具意义。 

学习辅导服务

AISG设有额外的学习帮助，服务于一些有轻微学习差别的学生。我们的EAL（英语作为额外语言）课程是为帮助学生更好地学习和取得学术成绩而设计。EAL学生在平常的课堂中能获得额外的语言协助。这是一种在常规课堂里插入内置式语言辅助服务的模式。. 

我们的学习辅助项目（Learning Support）与课堂协助相结合，通过小团队指导为需要额外辅助的学生提供个性化服务。学习辅导项目的总目标是帮助有需要的学生发展学术技能，让他们能健康地成长为独立的学习者。
